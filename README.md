# termfish

a python tracker that reads your webfishing file and makes a representation of your progress in the terminal !

handy when combined with [webfishing journal calculator](https://docs.google.com/spreadsheets/d/1qlwYDcNzLk8AZFGccNWA126RnYcYhQF6uPV3k0i3lrU/edit?gid=1755573711#gid=1755573711)

## install

```bash
git clone https://github.com/dootss/termfish.git
cd termfish
pip install -r requirements.txt
python termfish.py
```
